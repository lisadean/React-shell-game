# React-shell-game

* Create a new React app
* You'll want to create a Game component and a Shell component
* The player will play in multiple rounds, choosing (clicking) one of three Shells
* If they pick the correct Shell, the get a point. If they choose the wrong Shell, they lose a turn.
* The player will have a total of three turns until they need to start the game over
* The Game will need to randomize which Shell is the "correct" one.
* After three correct answers in a row, increase the number of Shells displayed to 5.
